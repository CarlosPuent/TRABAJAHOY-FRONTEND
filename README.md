# Portal-de-Trabajo-TRABAJAHOY

Frontend estático del portal TrabajaHoy construido con HTML y CSS.

## Estructura

- `src/images/` recursos compartidos
- `views/login/` vista de login
- `views/register-candidate/` registro de candidato
- `views/register-company/` registro de empresa
- `views/landing/` landing page

## Tecnologías

- HTML5
- CSS3
